The Developement Environment used in this project is NetBeans. This IDE was chosen due to ease of compatibility with Java Swing. The download link for netBeans can be found at the following link:
https://netbeans.org/downloads/

For DataBase Management, we chose to use MySQL Command Line Client and used an Wizard-based MySQL Installer.
The Installer can be found on the link http://dev.mysql.com/downloads/windows/installer/

To enable database device drivers for connectivity with Netbeans (the Development Environment used in this project),
we installed jdbc (J 5.1.34) connector which can be found at the link http://dev.mysql.com/downloads/connector/j/

A mysql-connector-java jar file can be found by unzipping the downloaded folder. We can then add this jar file to the Libraries package of project by following instructions on this tutorial: 
http://jsumon.wordpress.com/2009/11/24/adding-external-jar-or-library-to-netbeans-java-project/
